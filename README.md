# svtRNA-annotation
svtRNA annotation resources and GFF files associated with published work.

## Contents

- `svtRNA_annotation_v1.gff3` — miRNA-like svtRNAs genome annotation file in GFF format.
- `svtRNA_plus_miRNAs_annotation_v22.gff3` — miRNA-like svtRNAs + miRBase miRNAs annotation (v22) file in GFF format.
- Supplementary information is available in the associated manuscript.

## Version

Current release: v1.0 (bioRxiv preprint version)

## Citation

If you use this annotation in your work, please cite the associated manuscript.
